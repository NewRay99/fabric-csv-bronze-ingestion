# BCT Power BI font guide

## Approved family

Use **Lato** for page titles, visual titles, KPI values, labels and body copy. This follows the current Birmingham Children's Trust website.

Use **Arial** only as the deployment-safe fallback when Lato is unavailable in Power BI Desktop, the Power BI service, an export renderer or a locked-down client device.

## Recommended styles

| Content | Family | Weight | Size |
|---|---|---:|---:|
| Page title | Lato | Bold | 24–30px |
| Section heading | Lato | Bold | 16–20px |
| Visual title | Lato | Bold | 12–14px |
| KPI value | Lato | Bold | 28–36px |
| KPI / axis label | Lato | Regular | 11–12px |
| Table body | Lato | Regular | 11–12px |
| Metadata | Lato | Regular | 10–11px |

Do not use Poppins or Inter in this report brand. Avoid all-caps paragraphs and excessive letter spacing. Test PDF and PowerPoint exports before release because font substitution can change line wrapping.

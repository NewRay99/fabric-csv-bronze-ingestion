# BCT Power BI design ideas

The SVG scorecards are branded, accessible design references with illustrative values:

- `kpi-scorecard-total-referrals.svg`
- `kpi-scorecard-active-referrals.svg`
- `kpi-scorecard-engagement-rate.svg`

Each card uses the approved terms **Current period**, **Previous month**, **Month-on-month change** and **Change rate**. Percentage-point changes are explicitly marked `pp`.
